인공지능 커리큘럼

Artificial Intelligence

1. Pandas, Numpy를 이용한 데이터 분석

데이터 전처리에 활용되는 데이터 분석 기법을 배웁니다.

2. Tensorflow 사용법

Tensorflow의 기초 사용법을 배웁니다.

3. 선형회귀 선형 회귀의 개념을 배우며, 선형회귀모델을 직접 구축합니다. ex. 모의고사 성적으로 수능 성적을 예측하는 모델 구축하기.

가중치, 비용함수, 경사하강법 등 인공지능의 기본 개념을 배웁니다.

4. Keras 딥러닝 모델 구축에 활용되는 Keras의 기초 사용법을 배웁니다. Keras로 다중선형회귀모델, 비선형회귀모델 구축하기.

ex. 미래 인구수 예측하는 모델 구축하기.

5. 딥러닝 모델 구축 Layer, Node, 활성함수, Optimizer 등 딥러닝의 기본 개념을 배웁니다.

딥러닝 모델을 직접 구축하는 방법을 배웁니다.

6. CNN 딥러닝으로 이미지나 영상 데이터를 처리하는 데 활용되는 CNN에 대해 배웁니다. ex. 손글씨 인식 프로그램 만들기. ex. 개 이미지와 고양이 이미지를 구분하는 프로그램 만들기.

ex. 제품 이미지 데이터 중 정상 이미지와 불량 이미지를 구별하는 프로그램 만들기.

7. 자연어 처리 인공지능을 이용한 자연어 처리 기법을 배웁니다. ex. 네이버 영화 리뷰 분류하기.

ex. 한국어 챗봇 만들기.

커리큘럼은 수강생의 요청에 따라 얼마든지 수정 가능합니다.

인공지능 커리큘럼은 Python을 이용하여 진행합니다.
